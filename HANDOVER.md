이 문서는 프로젝트를 이어받거나, 나중에 다시 손볼 때 빠르게 파악하기 위한 안내입니다.

| 항목 | 내용 |
|------|------|
| 프로젝트 | AI 탁구로봇과 탁구 대결 (1인용 웹 탁구 게임) |

## 2. 파일 구조

```
pingpong/
├── index.html      ← 게임 본체 (이 파일이 전부)
├── PRD.md          ← 제품 요구사항 문서
├── HANDOVER.md     ← (이 문서) 인수인계
└── README.md       ← 프로젝트 소개
```

`index.html` 내부 구조:
- `<style>` — 모든 CSS (레이아웃, 로봇 캐릭터, 말풍선, 난이도 팝업, 반응형)
- `<body>` — 제목, 로봇 캐릭터(SVG), 캔버스, 난이도 팝업(overlay), 조작 안내
- `<script>` — 게임 로직 전체 (IIFE로 감싸 전역 오염 방지)

## 3. 자주 바꾸게 될 설정값

모두 `<script>` 상단 **"설정값"** 주석 구역에 모여 있습니다.

| 상수 | 현재값 | 의미 |
|------|--------|------|
| `WIN_SCORE` | 7 | 승리 점수 |
| `SMASH_MULTIPLIER` | 1.9 | 스매시 시 속도 배수 |
| `SMASH_SPEED_MAX` | 22 | 스매시 최고 속도 상한 |
| `BALL_SPEED_MAX` | 14 | 일반 타격 속도 상한 |
| `RAMP_PER_SECOND` | 0.05 | 랠리 중 초당 가속 비율 |
| `RAMP_MAX` | 1.8 | 가속 배율 상한 |

**난이도** (`DIFFICULTIES` 객체):

| 난이도 | aiSpeed (AI 패들 속도) | ballSpeed (공 초기 속도) | smashChance (AI 스매시 확률) |
|--------|-----|-----|-----|
| 쉬움 | 5.5 | 6 | 0 (안 함) |
| 보통 | 6 | 6 | 0.12 |
| 어려움 | 8 | 7 | 0.25 |

> 쉬움이 너무 쉬우면 `aiSpeed`를 올리고, 너무 어려우면 내리세요.
> 단, 보통(6)과 너무 가까워지면 난이도 구분이 사라집니다.
> `smashChance`는 AI가 공을 받아칠 때마다 스매시할 확률입니다. 너무 자주 맞으면 내리세요.
> AI는 랠리가 `AI_SMASH_MIN_RALLY`(기본 3)회 이상 이어진 뒤부터 스매시를 시도합니다.

## 4. 기능별 코드 위치 (수정 가이드)

- **AI 로봇 멘트 추가/수정** → `TAUNTS` 객체. 카테고리별(`start`, `aiScore`, `playerScore`, `rally`, `aiWin`, `playerWin`) 배열에 문장을 추가하면 됨. 랜덤으로 하나가 선택됨.
- **로봇 캐릭터 모양** → `<body>`의 `<svg aria-label="AI 탁구로봇">` 부분 (안테나·머리·눈·입).
- **파워업 아이템** → `ITEM_TYPES` 배열과 `applyItem()` / `expireEffect()` 함수. 효과 지속시간은 `applyItem()` 안의 숫자(초).
- **사운드** → `sfx` 객체. 음원 파일 없이 `beep(주파수, 길이, 파형)`으로 즉석 생성.
- **장애물** → `obstacle` 객체(위치·크기·속도)와 `update()` 내 충돌 처리.
- **기록 저장** → `localStorage` 키 `pp_bestRally`(최고 랠리), `pp_winStreak`(연승).

## 5. 조작법

| 입력 | 동작 |
|------|------|
| 마우스 상하 이동 / ↑ ↓ | 패들 이동 (창 전체에서 추적) |
| 마우스 클릭 / Shift | 스매시 (공 맞는 순간 누르고 있어야 발동) |
| Space | 시작 / 일시정지 / 재시작 |
| M | 음소거 토글 |
| 1 / 2 / 3 | 난이도 선택 (대기·종료 화면에서) |

## 6. 배포·업데이트 방법

이미 Netlify에 배포되어 있습니다(https://pingpong26.netlify.app/).

**최신 버전으로 업데이트하려면:**
1. https://app.netlify.com 로그인
2. `pingpong26` 사이트 선택 → **Deploys** 탭
3. `pingpong` 폴더(또는 `index.html`)를 드래그&드롭
4. 자동 배포되어 같은 주소에 반영

> 빠른 재배포만 필요하면 https://app.netlify.com/drop 에 폴더를 다시 드롭해도 됩니다.

## 7. 알려진 제약 / 다음 개선 후보

- **AI 스매시** — 난이도별 확률(`smashChance`)로 AI도 강타함. 시각 효과는 플레이어 스매시와 동일(노란 잔광). 색을 달리하려면 `bounceFrom`의 스매시 색을 타구 주체별로 분기하면 됨.
- **2인 모드 없음** — 현재 1인 전용. 우측 패들을 두 번째 플레이어가 조작하게 확장 가능.
- **모바일 조작** — 터치로 패들·스매시 지원하나, 작은 화면에서는 난이도 팝업 버튼이 다소 큼.
- **사운드 자동재생 정책** — 브라우저 정책상 첫 사용자 입력(클릭/Space) 후에 소리가 켜짐. 정상 동작이며 버그 아님.
