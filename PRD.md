# 📄 PRD — AI 탁구로봇과 탁구 대결

> 1인용 웹 탁구 게임. AI 탁구로봇을 상대로 즐기는 캐주얼 아케이드 게임.
> 배포 주소: https://pingpong26.netlify.app/

---

## 1. 프로젝트 개요

- **해결하려는 문제**
  탁구를 좋아해서, 직접 즐길 수 있는 1인용 탁구 게임을 만들어보고 싶다.
  (학습·재미 목적의 토이 프로젝트)

- **배경**
  평소 탁구를 좋아하고, 직접 게임으로 구현해보고 싶은 마음에서 시작.

- **기대 변화**
  없음 → 내가 만든 게임으로 실제로 플레이할 수 있게 된다.
  (나아가 웹에 배포해 누구나 링크로 접속해 즐길 수 있게 됨)

---

## 2. 사용자 분석

- **주 사용자**
  게임을 직접 만든 본인. (1인 제작·1인 플레이)
  + 링크를 공유받은 친구·동료도 동일하게 플레이 가능.

- **사용 맥락**
  PC에서, 작업 중 쉬는 시간에 잠깐 켜서 가볍게 플레이.
  웹 배포되어 모바일·태블릿에서도 접속 가능(반응형).

- **현재 해결 방식과 한계**
  지금은 시중 게임으로 즐김. 다만 "내가 직접 만든 게임으로 플레이"하는
  경험은 채워지지 않음. → 이 프로젝트로 충족.

---

## 3. 기능 요구사항

### 핵심 기능 (우선순위 순)

1. **탁구 게임 플레이** — 마우스 또는 ↑↓ 키로 패들을 움직여 AI 상대와 공을 주고받음. 먼저 7점을 내면 승리.
2. **스매시** — 공이 패들에 맞는 순간 마우스 클릭(또는 Shift)을 누르고 있으면 강하게 강타. 공이 노랗게 빛나며 빠르게 날아감.
3. **난이도 선택** — 시작 팝업에서 쉬움/보통/어려움 선택. AI 속도와 공 속도가 달라짐.

### 부가 기능 (재미 요소)

- **AI 탁구로봇 캐릭터 + 도발 멘트** — 상단 로봇이 상황별 멘트를 말풍선으로 표시 (시작/득점/실점/긴 랠리/승패).
- **파워업 아이템** — 경기 중 `?` 아이템 등장. 공이 닿으면 패들 확대 / 느린 공 / 상대 패들 축소 효과.
- **움직이는 장애물** — 가운데에서 위아래로 움직이는 벽. 공이 부딪히면 튕김.
- **점점 빨라지는 모드** — 랠리가 길어질수록 공이 점점 빨라짐.
- **기록 시스템** — 최고 랠리·연승을 브라우저에 저장(다음 접속 시 유지).
- **연출** — 사운드 효과, 타격 시 화면 흔들림·파티클, 공 잔상.

### 차별점

단순 Pong에 머무르지 않고 **AI 로봇 캐릭터의 도발 멘트**로 대결 분위기를 살린 점.
설치·계정 없이 링크 하나로 바로 즐길 수 있음.

---

## 4. 비기능 요구사항

- **사용 환경**
  웹 브라우저(데스크톱 기본, 모바일·태블릿 반응형 지원).
  단일 HTML 파일로 별도 설치·서버 불필요.

- **데이터 민감도**
  개인정보·결제 정보 없음. 게임 기록(최고 랠리·연승)만 브라우저 localStorage에 로컬 저장.

- **제약**
  1인 개발. 외부 라이브러리·이미지 파일 의존성 없음(바닐라 JS + Canvas + SVG).
  로봇 캐릭터는 SVG로 직접 그려 파일 의존성 제거.

---

## 5. 성공 지표

- **성공 모습**
  내가 만든 게임으로 실제로 플레이할 수 있고, 웹에 배포되어 링크로 누구나 접속 가능. ✅ 달성

- **핵심 지표**
  - 1차(달성): 동작하는 게임 완성 + 웹 배포(https://pingpong26.netlify.app/)
  - 2차(선택): 공유한 사람들이 실제로 플레이하고 "한 판 더" 하게 되는 재미

---

## 부록: 사용자 여정 (User Journey)

1. 링크 접속 → 난이도 선택 팝업 등장
2. 쉬움/보통/어려움 선택 후 `시작 ▶`
3. 마우스로 패들 조작, 공이 올 때 클릭으로 스매시
4. AI 로봇이 득점·실점마다 도발 멘트
5. 7점 선취 시 승패 표시 → 난이도 다시 골라 재도전
6. 최고 랠리·연승 기록이 누적되어 다음 접속 시에도 유지
