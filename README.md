# 🏓 AI 탁구로봇과 탁구 대결

AI 탁구로봇을 상대로 즐기는 **1인용 웹 탁구 게임**.
도발하는 로봇을 이겨보세요!

**▶ 지금 플레이:** https://pingpong26.netlify.app/

---

## ✨ 특징

- 🏓 **탁구 대결** — 마우스나 키보드로 패들을 움직여 AI와 랠리. 먼저 7점 내면 승리
- ⚡ **스매시** — 공 맞는 순간 클릭하면 강하게 강타 (노랗게 빛남)
- 🎚️ **난이도 선택** — 쉬움 / 보통 / 어려움
- 🤖 **도발하는 AI 로봇** — 상황에 따라 "이건 못 받겠지?", "인간~ 쫌 하는데?" 등 멘트
- 🎁 **파워업 아이템** — 패들 확대 / 느린 공 / 상대 패들 축소
- 🧱 **움직이는 장애물** — 가운데 벽에 공이 튕김
- 🔥 **점점 빨라지는 모드** — 랠리가 길수록 공 가속
- 🏆 **기록 저장** — 최고 랠리·연승이 브라우저에 저장됨
- 🔊 **사운드 & 연출** — 효과음, 화면 흔들림, 파티클, 공 잔상
- 📱 **반응형** — PC·모바일·태블릿 모두 지원

## 🎮 조작법

| 입력 | 동작 |
|------|------|
| 마우스 상하 / `↑` `↓` | 패들 이동 |
| 마우스 클릭 / `Shift` | 스매시 |
| `Space` | 시작 / 일시정지 / 재시작 |
| `M` | 음소거 |
| `F` | 전체화면 전환 (우상단 ⛶ 버튼으로도 가능) |
| `1` `2` `3` | 난이도 선택 |

## 👥 2인 온라인 대전

서로 다른 기기·네트워크에서 **방 코드**로 매칭해 실시간으로 대전합니다.

**▶ 플레이:** 시작 화면의 "👥 친구와 2인 온라인 대전" 링크, 또는 https://pingpong26.netlify.app/versus.html

1. 한 명이 **방 만들기** → 생긴 4자리 코드를 상대에게 공유
2. 상대가 그 코드를 입력 → **입장하기** → 바로 대전
3. 양쪽 모두 마우스·터치·`↑` `↓`로 조작, 게이지가 차면 자동 스매시, 먼저 7점 승리

- **호스트 권한 방식**: 방장 브라우저가 물리를 계산하고, 서버는 방 매칭 + 메시지 중계만 담당(게임 로직 모름).
- **서버**: 별도 저장소 [`nsg8739/pingpong-server`](https://github.com/nsg8739/pingpong-server) (Node + `ws`), **Render** 호스팅. `git push` 시 자동 재배포.
- ⚠️ 무료 서버라 첫 접속에 **30~60초 콜드스타트**가 있을 수 있고, **방장이 탭을 전환/최소화하면** 양쪽 게임이 잠시 멈춥니다(구조상 한계).

## 🛠️ 기술 스택

- **바닐라 JavaScript** (프레임워크 없음)
- **HTML5 Canvas 2D** — 게임 렌더링
- **Web Audio API** — 음원 파일 없이 효과음 생성
- **localStorage** — 기록 저장
- **SVG** — 로봇 캐릭터 (이미지 파일 없음)

> 전체가 단일 `index.html` 파일이며, 외부 라이브러리·빌드 도구·이미지 의존성이 없습니다.
