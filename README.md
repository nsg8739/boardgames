# 🏓 AI 탁구로봇과 탁구 대결

AI 탁구로봇을 상대로 즐기는 **1인용 웹 탁구 게임**.
도발하는 로봇을 이겨보세요!

**▶ 지금 플레이:** https://pingpong26.netlify.app/

---

## ✨ 특징

- 🏓 **탁구 대결** — 마우스나 키보드로 패들을 움직여 AI와 랠리. 먼저 7점 내면 승리
- ⚡ **스매시** — 공 맞는 순간 클릭하면 강하게 강타 (노랗게 빛남)
- 🎚️ **난이도 선택** — 쉬움 / 보통 / 어려움
- 🤖 **도발하는 AI 로봇** — 상황에 따라 "이건 못 받겠지?", "인간~ 쫌 하는데?" 등 멘트
- 🎁 **파워업 아이템** — 패들 확대 / 느린 공 / 상대 패들 축소
- 🧱 **움직이는 장애물** — 가운데 벽에 공이 튕김
- 🔥 **점점 빨라지는 모드** — 랠리가 길수록 공 가속
- 🏆 **기록 저장** — 최고 랠리·연승이 브라우저에 저장됨
- 🔊 **사운드 & 연출** — 효과음, 화면 흔들림, 파티클, 공 잔상
- 📱 **반응형** — PC·모바일·태블릿 모두 지원

---

## 🎮 조작법

| 입력 | 동작 |
|------|------|
| 마우스 상하 / `↑` `↓` | 패들 이동 |
| 마우스 클릭 / `Shift` | 스매시 |
| `Space` | 시작 / 일시정지 / 재시작 |
| `M` | 음소거 |
| `F` | 전체화면 전환 (우상단 ⛶ 버튼으로도 가능) |
| `1` `2` `3` | 난이도 선택 |

## 🚀 실행 방법

별도 설치가 필요 없습니다.

- **온라인**: https://pingpong26.netlify.app/ 접속
- **로컬**: `index.html` 파일을 더블클릭하면 브라우저에서 바로 실행

---

## 🛠️ 기술 스택

- **바닐라 JavaScript** (프레임워크 없음)
- **HTML5 Canvas 2D** — 게임 렌더링
- **Web Audio API** — 음원 파일 없이 효과음 생성
- **localStorage** — 기록 저장
- **SVG** — 로봇 캐릭터 (이미지 파일 없음)

> 전체가 단일 `index.html` 파일이며, 외부 라이브러리·빌드 도구·이미지 의존성이 없습니다.
